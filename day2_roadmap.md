# Rust Day 2 Learning Roadmap 🦀

Now that you understand variables, mutability, functions, and printing, Day 2 is all about handling data and controlling how your program runs.

## 📋 Day 2 Checklist

- [ ] **1. Data Types in Rust**
  - Scalar types: Integers (signed/unsigned like `i32`, `u32`), Floats (`f64`), Booleans (`bool`), and Characters (`char`).
  - Compound types: Tuples `(x, y)` and Arrays `[1, 2, 3]`.
- [ ] **2. Variable Shadowing**
  - What is shadowing? (Declaring a new variable with the same name as a previous one).
  - How does shadowing differ from mutability?
- [ ] **3. Control Flow**
  - `if` / `else if` / `else` expressions.
  - Learning that `if` is an expression (it can return a value!).
- [ ] **4. Loops in Rust**
  - The `loop` expression (infinite loop with `break`).
  - The `while` loop.
  - The `for` loop (iterating through a range or an array).

## 🎯 Target Practice Challenges for Today

Here are three simple programs you will write today to test your knowledge:

### 💻 Challenge 1: The Temperature Converter
Create a program that converts temperatures between Fahrenheit and Celsius.
- *Learnings tested: Data types (floats), basic math, and simple functions.*

### 💻 Challenge 2: Fibonacci Generator
Generate the $n$-th Fibonacci number.
- *Learnings tested: Control flow, loops (`for` or `while`), and mutability.*

### 💻 Challenge 3: Array Iterator
Define an array of numbers and print only the even numbers using a `for` loop.
- *Learnings tested: Arrays, `for` loops, and `if` conditions.*
